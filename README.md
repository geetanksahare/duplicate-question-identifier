# duplicate-question-identifier
An NLP project that identifies duplicate questions using text preprocessing, similarity features, and machine learning models.

# Duplicate Question Detection (NLP)

This project detects whether two questions are duplicates using NLP and Machine Learning.

## Features
- Text preprocessing (NLTK)
- Feature engineering & similarity metrics
- ML models (Random Forest, XGBoost)
- Duplicate question prediction

## Tech Stack
- Python
- Scikit-learn
- NLTK
- XGBoost
- Pandas, NumPy

## How to Run
1. Clone repo
2. Install requirements
3. Open notebook

## Dataset
Quora Question Pairs dataset
